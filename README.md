# BAC Tunisia — Data Mining Project
### Socioeconomic Determinants of BAC Success Across 26 Governorates (2018–2025)

---

## Project Overview

This project applies data mining techniques to the Tunisian BAC (Baccalauréat) exam results dataset, covering all 26 governorates from 2018 to 2025. It goes beyond descriptive analysis to discover hidden patterns — non-obvious groupings, threshold effects, unexpected performers, temporal trajectories, and convergence dynamics.

**Type of data:** Panel data (hybrid). Each governorate has 8 yearly observations. This is NOT classical time series (too short for ARIMA/LSTM), but it IS rich enough for trend analysis, trajectory clustering, change point detection, and panel regression with fixed effects.

**Project Goal:** Understand which socioeconomic factors (poverty, illiteracy, school infrastructure) best predict BAC success rates, identify regional clusters with distinct performance patterns, and detect temporal trends that reveal whether educational inequality is increasing or decreasing.

## Dataset

| File | Description |
|------|-------------|
| `data/raw/bac_2018_2025.csv` | Raw BAC results — wide format, one row per governorate (Arabic names) |
| `data/raw/ecoles.csv` | School infrastructure data (6109 schools across all governorates) |
| `data/raw/socio_economic.csv` | Socioeconomic indicators: poverty rate, illiteracy rate, population |
| `data/clean/bac_panel.csv` | Generated by NB02 — long format, 208 rows (26 × 8 years) |
| `data/clean/bac_crosssectional.csv` | Generated by NB02 — 26 rows with engineered features |
| `data/enriched/enriched_bac_data.csv` | Generated by NB01 — enriched dataset with all features merged |

**Raw features per governorate:** BAC rank, number of candidates, number who passed, success rate (per year 2018–2025), plus static: number of lycées (high schools), poverty rate, illiteracy rate, population.

**Note on Sfax and Tunis:** Sfax and Tunis are each split into two exam centers (1 and 2) that share the same socioeconomic features. They are flagged with `is_split_gov = True` in the panel dataset.

## Notebooks

### NB01 — Data Integration & Enrichment
**Run this first.** Loads and merges three raw data sources into a single enriched dataset.

**Inputs:**
- `data/raw/bac_2018_2025.csv` — BAC results (Arabic names, percentage strings with commas)
- `data/raw/ecoles.csv` — school infrastructure (6109 schools across all governorates)
- `data/raw/socio_economic.csv` — poverty rate, illiteracy rate, population per governorate

**Key steps:**
- Parses percentage strings (`"64,36%"` → `64.36`)
- Maps Arabic governorate names to French for consistency
- Aggregates school counts by type (primaire, collège, lycée) per governorate
- Merges all three sources on `gov_fr` (French governorate name)
- Computes normalized features: `lycees_per_1k` (normalized by estimated 15–19 population), trend slope, volatility, COVID recovery ratio
- Handles split governorates (Sfax 1/2, Tunis 1/2) which share identical socioeconomic features

**Output:** `data/enriched/enriched_bac_data.csv` (26 rows, one per governorate with all features)

---

### NB02 — Data Preparation & EDA
Reshapes the enriched data into panel and cross-sectional formats, engineers features, and produces first-level insights.

**Input:** `data/enriched/enriched_bac_data.csv`

**Key steps:**
- Reshapes wide format (1 row per governorate) → long panel format (208 rows: 26 governorates × 8 years)
- Flags split governorates (Sfax 1/2, Tunis 1/2) with `is_split_gov = True`
- Engineers cross-sectional features: `mean_rate`, `trend_slope`, `std_rate`, `covid_recovery_pct`, `rank_volatility`, `best_rank`, `worst_rank`
- Computes COVID recovery percentage: `(2021_rate - 2020_rate) / 2019_rate * 100`
- Validates data integrity and checks for missing values

**Outputs:**
- `data/clean/bac_panel.csv` — 208 rows, columns: `governorate, code, year, rank, success_rate, nb_lycees, lycees_per_1k, poverty_rate, illiteracy_rate, population, is_split_gov`
- `data/clean/bac_crosssectional.csv` — 26 rows with all engineered features

---

### NB03 — Clustering & Pattern Discovery
The core **data mining** notebook. Discovers hidden structure using unsupervised learning.

**Inputs:** `data/clean/bac_crosssectional.csv`, `data/clean/bac_panel.csv`

**Key analyses:**
- Elbow + silhouette analysis → optimal k = 2 clusters
- K-Means clustering on `[poverty_rate, illiteracy_rate, lycees_per_1k, mean_rate, trend_slope, rank_volatility]`
- **Cluster 0 (coastal/developed):** Sfax 1&2, Monastir, Mahdia, Sousse, Ariana, Nabeul, Tunis 1&2, Médenine, Ben Arous, Bizerte, Gabès, Mannouba — avg poverty ~11%, avg BAC rate ~45%
- **Cluster 1 (interior/disadvantaged):** Zaghouan, Siliana, Béja, Kef, Kairouan, Kasserine, Sidi Bouzid, Tozeur, Tataouine, Kébili, Gafsa, Jendouba — avg poverty ~26%, avg BAC rate ~30%
- Hierarchical clustering dendrogram
- Threshold effect detection (poverty/illiteracy breakpoints)
- Rank mobility analysis (2018→2025)

**Key finding:** Two natural clusters align closely with the coastal/interior geographic divide, with poverty and illiteracy as the primary separating features.

---

### NB04 — Panel & Temporal Analysis
Exploits the time dimension to find temporal patterns and detect convergence/divergence trends.

**Inputs:** `data/clean/bac_panel.csv`, `data/clean/bac_crosssectional.csv`

**Key analyses:**
- Per-governorate linear trend slopes with statistical significance (p < 0.05)
- **Statistically significant improvers:** Tataouine (slope +3.63, p=0.008), Kébili (slope +2.66, p=0.016)
- All other governorates: stable trends (not statistically significant at p < 0.05)
- Year-over-year change heatmap (change point detection)
- Beta-convergence test — are weaker regions catching up?
- Year fixed effects panel regression

**Key question answered:** Is educational inequality between Tunisian regions increasing or decreasing over time?

---

### NB05 — Predictive Modeling & Feature Importance
Trains regression models and extracts which socioeconomic factors drive BAC success.

**Inputs:** `data/clean/bac_crosssectional.csv`, `data/clean/bac_panel.csv`

**Features used:** `poverty_rate`, `illiteracy_rate`, `lycees_per_1k`, `population`

**Model comparison (Leave-One-Out cross-validation, n=26):**

| Model | LOO R² | LOO MAE (%) |
|-------|--------|-------------|
| Gradient Boosting | 0.738 | 3.539 |
| Random Forest | 0.680 | 4.236 |
| Ridge (α=1) | 0.614 | 4.750 |
| Lasso (α=0.5) | 0.602 | 4.869 |
| Linear Regression | 0.589 | 4.938 |

**Key outputs:**
- Feature importance across 4 methods (RF impurity, RF permutation, Gradient Boosting, Ridge coefficients)
- Panel regression with year controls (208 obs)
- Predicted vs actual scatter with residual labeling
- **Policy simulation:** What would Kasserine's rate be if illiteracy dropped to the national median?
- Trained model saved as `models/bac_predictor.pkl`

**Expected finding:** Illiteracy rate is likely the strongest single predictor, stronger than poverty. School capacity per capita matters less than expected once poverty is controlled for.

## Data Mining vs Analysis — What's the Difference Here

This project deliberately goes beyond descriptive analysis:

| Analysis | Data Mining |
|----------|-------------|
| Sfax has the highest rate | Sfax is an overperformer: its rate exceeds what its poverty level predicts |
| Kasserine has the lowest rate | Kasserine belongs to a cluster where performance collapses beyond a poverty threshold |
| COVID caused a drop in 2020 | COVID hit structurally weaker regions harder — recoveries are correlated with poverty |
| Success rates vary by region | Trajectory clustering reveals groups that share patterns regardless of geographic location |
| Illiteracy correlates with BAC | Feature importance shows illiteracy dominates poverty in predictive power |

---

## Key Findings Summary

1. **Two Natural Clusters:** Coastal/developed vs interior/disadvantaged regions, separated primarily by poverty and illiteracy rates
2. **Illiteracy > Poverty:** Illiteracy rate is a stronger predictor of BAC success than poverty rate
3. **Stable Trends:** Most regions show stable performance over 2018–2025, with only Tataouine and Kébili showing statistically significant improvement
4. **COVID Impact:** 2020 saw a structural drop; recovery patterns varied by region
5. **No Convergence:** Weaker regions are not catching up to stronger ones; inequality remains stable
